# MVP Spec

## Personas
- Member: browses, posts needs, volunteers to help
- Organization Poster: posts opportunities on behalf of a church/school/nonprofit
- Moderator/Admin: handles flags, verifies orgs, keeps hub safe

## MVP Features
### Bulletin Board
- List/filter posts (type, category, status, time)
- View post detail
- Create/edit/close a post

### Volunteer Commitments
- Volunteer offers help
- Poster accepts/declines
- Private location/details revealed on acceptance
- Mark completed

### Messaging (minimal)
- Thread between post author and accepted volunteer(s)

### Moderation
- Flag content/users/orgs
- Admin UI for moderation queue
- Verify orgs
- Ban/suspend users
- Remove/hide posts/messages

## Non-goals (MVP)
- Federation
- Recurring classes with enrollment/attendance
- Maker space resource booking
- Payments
